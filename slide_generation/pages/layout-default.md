# What is Marp?

Marp (Markdown Presentation Ecosystem) is a minimalist presentation tool that converts Markdown into beautiful presentations. Here are its key features:

- ✍️ **Pure Markdown** - Create slides entirely in Markdown without complex formatting
- 🎭 **Built-in Themes** - Choose from clean, professional themes or create your own
- ⚡ **Fast & Simple** - Focus on content with minimal setup and configuration
- 🔌 **VS Code Integration** - First-class support in Visual Studio Code
- 👁️ **Live Preview** - See your slides update in real-time as you write
- 💾 **Multiple Formats** - Export to PDF, PowerPoint, or HTML slideshows
- 🔧 **Extensible** - Add custom CSS and HTML when needed
