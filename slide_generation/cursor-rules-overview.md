## **Key Sections Covered:**

1. **Frontmatter Syntax** - YAML configuration for slides including themes, transitions, layouts, etc.
2. **Slide Separators** - Using --- to separate slides
3. **Layouts** - Different layout types (default, two-cols, image-right, center)
4. **Vue Directives** - All the interactive features:
- v-click for click animations
- v-motion for motion animations
- v-mark for highlighting elements
- v-drag for draggable elements
1. **Code Blocks** - Various code block types:
- Basic highlighting
- Line highlighting with {1|3|5-7}
- Monaco editor with {monaco}
- Monaco runner with {monaco-run}
- TwoSlash support
- Magic move for code transitions
1. **Built-in Components** - Toc, Tweet, Youtube, Arrows, etc.
2. **Diagrams** - Mermaid and PlantUML support
3. **LaTeX Support** - Inline and block math with animations
4. **CSS and Styling** - Slide-scoped styles, UnoCSS classes
5. **Vue Script Setup** - Adding reactive data to slides
6. **Event Handling** - Navigation and interaction events
7. **Icons** - Carbon icon usage
8. **Notes** - Presenter notes syntax
9. **Best Practices** - Guidelines for effective Slidev usage

The rules document provides practical examples for each syntax pattern, making it easy to reference when creating Slidev presentations. It covers everything from basic slide structure to advanced interactive features like animations, code editing, and diagram generation.
